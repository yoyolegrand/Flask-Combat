# Flask Combat

Ce dépôt contient la spécification du mod Minecraft Forge **Flask Combat** destiné à la version 1.20.1 (Forge 47.4.0). L'objectif est d'introduire des fioles jetables infligeant des dégâts magiques et laissant des zones d'effet temporaires amplifiées par deux attributs personnalisés : **Flask Effectiveness** et **Flask Duration**.

- Consultez [`docs/FLASK_COMBAT_SPEC.md`](docs/FLASK_COMBAT_SPEC.md) pour l'arborescence complète, les formules de calcul et les exemples d'implémentation (Fire Flask, gestion des zones persistantes, attributs, crafting, etc.).
- Le document couvre également les exigences Forge (mods.toml, pack.mcmeta, DeferredRegister) ainsi que la roadmap de développement.

Ce dépôt sert de point de départ conceptuel avant la mise en œuvre du code source ForgeGradle.
